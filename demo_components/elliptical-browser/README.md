# Elliptical Browser

Browser Implementation of Node Express style  apps




# Installation

##npm

``` bash

git clone https://github.com/ellipticaljs/browser.git
npm install

```

##bower

``` bash

bower install elliptical-browser

```

##demo

``` bash

git clone https://github.com/ellipticaljs/browser.git
npm install
bower install
gulp demo
python -m SimpleHTTPServer 8080


```
